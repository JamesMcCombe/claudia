## ðŸ“Š PATTERNS ACROSS ALL 10

### Universal Winning Elements
1. **Hero CTA above fold** (100% of sites)
2. **Social proof within 2 scrolls** (logos, user count, reviews)
3. **Mobile-first design** (especially LATAM sites)
4. **Trust signals everywhere** (security, privacy, verification)
5. **Clear value proposition** (5-second understanding)
6. **Free tier or trial** (remove barrier to entry)
7. **Visual hierarchy** (image â†’ price â†’ details)
8. **Conversational copy** (you, your, we - not corporate)

### Color Palette Insights
- **Warm > Cold:** Coral, pink, orange outperform blue, gray for consumer trust
- **Gradients:** Smooth transitions feel premium (Nubank, Headspace)
- **White space:** Essential for readability on mobile
- **Accent colors:** Gold for badges (savings, premium), green for success

### Typography Trends
- **Sans-serif dominance:** Only Duolingo uses playful font (we'll stick to Poppins/Inter)
- **Large sizes:** 40-50px headlines (mobile-first)
- **Sentence case:** Not ALL CAPS (more approachable)
- **Bold weights:** 600-700 for headings (attention-grabbing)

### Animation Philosophy
- **Subtle > Flashy:** Fade-ups, slides, not spins/flips
- **Scroll-triggered:** Appear when visible, not all at once
- **Micro-interactions:** Button hovers, card lifts (delightful, not distracting)
- **Performance:** Never sacrifice load time for animation
